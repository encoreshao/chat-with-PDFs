# Chat with PDFs: RAG with LangChain, GPT & LLaMa in Python

## Packages

- OpenAI
- LangChain
- LangChain OpenAI
- LangChain Ollama
- Faiss

## Eveniment Setup

- pyenv virtualenv --version
- pyenv virtualenvs
- pyenv virtualenv chat-with-pdfs
- pyenv activate chat-with-pdfs --verbose
- pip install -r requirements.txt

## How to run

```
python src/main.py
```
